# **App Name**: MAKI e-commerce

## Core Features:

- Product Listing: Display all available appliances with their prices (e.g., Amplifier $350, Guitar $150).
- User Registration: Allow visitors to register and become customers.
- Shopping Cart: Enable users to add products to a shopping cart.
- Delivery Options: Let users choose their preferred delivery method.
- Location Selection: Enable users to select a delivery shop location (Bukavu, Uvira, Kinshasa, Goma).
- Order Confirmation: Finalize the order and display a summary for confirmation.
- Payment Gateway: Integrate multiple online payment options (Airtel money, orange money, m-pesa, VISA, MasterCard, PayPal, Bitcoin).

## Style Guidelines:

- Primary color: A vibrant orange (#FF7800) to convey energy and excitement.
- Background color: A light gray (#F0F0F0) for a clean and modern feel.
- Accent color: A deep blue (#003399) for important actions and elements, providing contrast.
- Body font: 'PT Sans', a humanist sans-serif suitable for body text
- Headline font: 'Playfair', a modern serif with an elegant, high-end feel; use 'PT Sans' for longer text.
- Use clear, recognizable icons for navigation and product categories.
- Maintain a clean, grid-based layout for product listings and information display.
- Use subtle transitions and loading animations to enhance user experience.
